# Contributing to PhotoMagic

Thank you for your interest in contributing to this educational security project!

## Purpose

This project exists to educate users about mobile privacy risks. All contributions should align with this educational mission.

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker
- Describe the issue clearly
- Include device/OS information if relevant

### Suggesting Enhancements
- Open an issue with the "enhancement" label
- Explain the educational value of the enhancement
- Describe expected behavior

### Code Contributions

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

### Code Style
- Use consistent indentation (2 spaces)
- Add comments for complex logic
- Follow React Native best practices

## Ethical Guidelines

All contributions must:
- Support the educational mission
- Not enable malicious use
- Include appropriate warnings
- Respect user privacy in documentation

## Questions?

Open an issue with the "question" label.
