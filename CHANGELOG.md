# Changelog

All notable changes to PhotoMagic will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-29

### Added
- Initial release of PhotoMagic InfoSec Demo
- Home screen with app branding and navigation
- Photo Editor screen with gallery/camera selection
- AI prompt input with suggestion chips
- Silent metadata extraction from photos
- EXIF data parsing (GPS, device info, timestamps)
- Real-time location harvesting with reverse geocoding
- Device fingerprinting
- Risk assessment engine with severity levels
- Hidden Security Reveal dashboard
- Privacy protection tips and recommendations
- Harvest history tracking
- Statistics summary view

### Security Features Demonstrated
- Permission abuse patterns
- EXIF metadata exposure
- Silent data collection techniques
- Location tracking capabilities
- Device fingerprinting methods

### Technical
- Built with React Native and Expo SDK 54
- React Navigation for screen management
- Context API for state management
- Android APK build support
